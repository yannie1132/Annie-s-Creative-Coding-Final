# Annie-s-Creative-Coding-Final

A touchless, interactive web experiment where users reveal hidden data by "wiping" a digital rain interface with their hands.

Features

Hand Tracking: Uses MediaPipe Hands to track palm and finger movements in real-time.


How to Run

Open Palm: Wipes away the digital noise to reveal hidden links.

Closed Fist: Acts as a "click" action to interact with revealed elements.



Libraries Used

p5.js - Creative coding framework.

MediaPipe Hands - Machine learning for hand tracking.
